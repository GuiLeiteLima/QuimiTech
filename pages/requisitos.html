Requisitos Mínimos:

Navegador: Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari (versões mais recentes). Processador: Processador dual-core de 1.6 GHz. Memória RAM: 2 GB. Placa Gráfica: Gráficos integrados ou equivalente. Conexão com a Internet: Conexão estável para carregar recursos online.



Requisitos Médios:

Navegador: Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari (versões mais recentes). Processador: Processador quad-core de 2.5 GHz. Memória RAM: 4 GB. Placa Gráfica: Placa dedicada com suporte para WebGL. Conexão com a Internet: Conexão estável para carregar recursos online.
